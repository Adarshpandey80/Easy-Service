# Easy-Service

Easy-Service is a full-stack MERN-based service marketplace platform that connects users with trusted local service providers for home maintenance and repair services. The platform enables users to discover services, book appointments, communicate with providers in real time, make secure online payments, and receive AI-powered assistance for troubleshooting and service recommendations.

## Key Features

* Secure JWT Authentication and Role-Based Access Control
* User, Shop Owner, Admin, and Super Admin Dashboards
* Service Listing, Booking, and Order Management
* Real-Time Chat System using Socket.io
* AI Assistant powered by Google Gemini for service recommendations and troubleshooting
* KYC Verification System for Service Providers
* Razorpay Payment Integration
* Ratings and Reviews System
* Cloudinary Image Upload and Management
* Responsive and User-Friendly Interface

## Tech Stack

**Frontend:** React.js, Redux Toolkit, Axios, CSS

**Backend:** Node.js, Express.js, MongoDB, Mongoose

**Authentication:** JWT, bcrypt

**Real-Time Communication:** Socket.io

**AI Integration:** Google Gemini, LangChain

**Payments:** Razorpay

**Media Storage:** Cloudinary

## Project Highlights

* Developed a scalable MERN-stack platform supporting multiple user roles and secure authentication.
* Implemented real-time messaging between users and service providers using Socket.io.
* Integrated AI-powered assistance to provide instant troubleshooting guidance and service recommendations.
* Built a complete service booking workflow with payment processing, KYC verification, and order tracking.
